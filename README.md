## Overview
A simple Q&A application with user authentication, categories, questions, answers, and comments.

## Features
- JWT auth: register, login, me 
- Category listing 
- Questions: list by category, create, delete 
- Answers: list by question, create, delete; comments add/delete 

## Technologies Used

### Presentation Layer
- HTML, CSS, & JavaScript
- React Framework + Vite

### Application Layer
- Express 5
- Mongoose 8
- bcrypt
- jsonwebtoken

### Data Layer
- MongoDB
